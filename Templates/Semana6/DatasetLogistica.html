Dataset Telecomunicaciones - Regresión Logística

Objetivo

Este dataset busca predecir si un cliente cancelará un servicio de telecomunicaciones basándose en tres variables: duración de la llamada, plan contratado y historial de pago.

Descripción

Se han generado 100 registros con datos simulados de clientes. La variable objetivo es la cancelación del servicio ("Sí" o "No").

Explicación de las Variables

  • Duración de la Llamada: Tiempo en minutos que el cliente habló con el servicio. Varía entre 1 y 60 minutos.
  • Plan Contratado: Indica si el cliente tiene un plan de suscripción (1 = Sí, 0 = No).
  • Historial de Pago: Representa qué tan puntual es el cliente con los pagos. Valores entre 0 y 1, donde 0 es un historial deficiente y 1 es un historial excelente.
  • Cancelación: Variable objetivo que indica si el cliente canceló el servicio ("Sí") o no ("No").
